# Python SQL Orders Analysis

## 📌 Project Overview
This project analyzes order data using Python, Pandas, and SQL.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- SQL Server
- Jupyter Notebook

## 📂 Dataset
- orders.csv

## 📊 Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- SQL Queries
- Data Visualization
- Business Insights

## 🚀 How to Run
1. Clone this repository.
2. Install Python libraries.
3. Open `orders data analysis.ipynb` in Jupyter Notebook.
4. Run all the cells.
